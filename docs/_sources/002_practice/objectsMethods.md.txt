# Объекты и методы

*Application Programming Interface (API)*
: Набор способов вызова процедур и связанных параметров, используемых для обеспечения взаимодействия между различным ПО. <p align="right"> *[ISO/IEC 18012-1:2004](https://www.iso.org/obp/ui#iso:std:iso-iec:18012:-1:ed-1:v1:en:term:3.1.1)* </p>

*API Google Earth Engine* состоит из ряда объектов и методов.
Объекты представляют собой определенные типы данных, которым доступны наборы специфических методов (процедур) обработки.
Объекты и методы комбинируются в виде пользовательских скриптов, которые передаются для обработки на серверы *Earth Engine*.

Данный раздел посвящен последовательному разбору основных объектов и методов, представленных в *API GEE*.
